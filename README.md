# cub3d

## Installation

to install:
```
git clone --recursive https://github.com/fgroo/cub3d.git
```

If `make` doesnt compile, make sure that the submodules `libft` and `MLX42` are
not empty. If they are please use this:

```
git submodule init
git submodule update --remote --recursive
```
